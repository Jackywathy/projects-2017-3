# Find It!
NCSS 2017, Group 3 - _**the best group**_
## Running from source
Clone the repo:
```bash
$ git clone https://github.com/ncss/projects-2017-3/
```
Create the database:
```bash
cd db/
python db_create.py
```
Start the server:
```bash
python server.py
```
The server runs by default at localhost:8888.
